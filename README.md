# Oshiwambo Translator App
The Oshiwambo Translator App is a mobile application designed to help people learn Oshikwanyama, one of the major dialects of the Oshiwambo language spoken in Namibia and parts of Angola.

This project was created to help bridge the accessibility gap for Oshiwambo speakers and learners, as very few digital translation tools exist for this language.

The app focuses on providing simple, fast translations of common words and phrases, making it useful for travellers, language learners, and members of the Namibian diaspora.

I began building this application because I will be travelling to Namibia, and I realised that there are almost no translation tools available for Oshiwambo.

While major languages have many digital resources, smaller languages are often underrepresented in technology.

This project is an attempt to create a simple but meaningful tool that makes communication easier while also learning how to build a complete mobile application from design to deployment.

# Features
- Translate English → Oshikwanyama
- Built-in dictionary-based translations
- Support for common conversational phrases
- Simple and intuitive mobile interface
- Designed for quick everyday communication

  # Technologies
- Flutter – cross-platform mobile development
- Dart – application logic
- SQLite / local storage – translation data
- Figma – UI design and prototyping
